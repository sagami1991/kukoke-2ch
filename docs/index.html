くこけ？2ch

新世代2ch専用ブラウザ
ダウンロード（近日リリース予定）

特徴

ウインドウ式UI

「くこけ？2ch」はよくある旧世代の分割画面ではなく、自由自在なウインドウ式を採用しています。

マルチプラットフォーム対応

Mac, Windows, Linux すべてに対応しています。

絵文字対応

現代のコミュニケーションにおいて、絵文字は欠かせないものの一つです。

シンプルかつ使いやすいUI

分からづらいボタンにはすべてラベルがついています。最近のアプリのように、何の操作か分からないということがありません。

フリーソフト

「くこけ？2ch」はフリーソフトで、広告もありません。オープンソースなので誰でも開発に参加可能です。
